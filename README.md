# Carmen Díaz — Portfolio

The personal portfolio website for Carmen Díaz, built with **Next.js**, **TypeScript**, and **Tailwind CSS**. It showcases software engineering experience, skills, and projects.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Deployment:** [Vercel](https://vercel.com/)

## 🚀 Getting Started

### 1. Install Dependencies
This project uses Bun as the package manager.
```bash
bun install
```

### 2. Run the Development Server
```bash
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Build for Production
```bash
bun run build
```

## 📂 Project Structure
- `/app` — Page routes and layouts.
- `/components` — Reusable UI components.
- `/data` — JSON data files for portfolio content (e.g., bio, experience).
- `/lib` — Utility functions and helper modules.
